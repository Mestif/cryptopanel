# Обновление версии 1.7.0

## Изменения

### 1. Добавлена пометка "(только SPOT)" к надписи "Добавить тикер"
- ✅ Текст изменен с "Добавить тикер вручную:" на "Добавить тикер вручную (только SPOT):"
- ✅ Уточняет, что поддерживаются только SPOT тикеры (не фьючерсы)
- ✅ Помогает пользователям понять ограничения

### 2. Убраны "Powered by binance" и "disclaimer" из виджета
- ✅ Удалены атрибуты `data-powered-by` и `data-disclaimer` из виджета Binance
- ✅ Виджет теперь отображается без дополнительных надписей
- ✅ Более чистый интерфейс

### 3. Заменена иконка приложения
- ✅ Новая иконка из `ICON.png` (563 KB)
- ✅ Иконка автоматически копируется при сборке
- ✅ Поддержка как `ICON.png`, так и `Icon.png`
- ✅ Иконка отображается в Finder, Dock и меню приложений

## Технические детали

### SettingsView
- Текст изменен на "Добавить тикер вручную (только SPOT):"
- Уточняет ограничение на SPOT тикеры

### BinanceWidgetView
- Удалены атрибуты:
  - `data-powered-by="Powered by"`
  - `data-disclaimer="Disclaimer"`
- Виджет теперь без дополнительных надписей

### build.sh
- Добавлена поддержка `ICON.png` (с заглавными буквами)
- Приоритет поиска:
  1. `ICON.png` в корне
  2. `Icon.png` в корне

  4. `CryptoPanelApp/Icon.png`
- Иконка копируется как `Icon.png` в Resources

## Использование

### Добавление тикеров
- Пометка "(только SPOT)" напоминает, что поддерживаются только SPOT тикеры
- Фьючерсы и другие типы контрактов не поддерживаются
- Используйте только тикеры из SPOT рынка Binance

### Виджет
- Виджет теперь без надписей "Powered by" и "Disclaimer"
- Более чистый и минималистичный интерфейс
- Фокус на данных о ценах

### Иконка
- Новая иконка с логотипами BTC, ETH, SOL
- Отображается во всех местах системы
- Автоматически применяется при сборке

## Файлы

- `SettingsView.swift` - добавлена пометка "(только SPOT)"
- `BinanceWidgetView.swift` - убраны надписи из виджета
