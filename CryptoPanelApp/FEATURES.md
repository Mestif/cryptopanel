# Функции CryptoPanel

## Основные возможности

### 1. Отображение цен в меню баре
- Вместо иконки отображаются цены выбранных криптовалют
- Формат: `BTC: $XX,XXX | ETH: $X,XXX | SOL: $XXX`
- Автоматическое обновление каждые 30 секунд
- Компактное отображение с моноширинным шрифтом

### 2. Виджет Binance (левая кнопка мыши)
- По левому клику на цены открывается виджет Binance
- Отображает таблицу с BTC, ETH, SOL
- Показывает:
  - Название монеты
  - Текущую цену
  - Изменение за 24 часа (%)
  - График цены
- Размер окна: 800x600 пикселей
- Темная тема

### 3. Меню настроек (правая кнопка мыши)
- По правому клику открывается меню настроек
- Выбор тикеров для отображения в меню баре
- Доступные тикеры:
  - BTC (Bitcoin)
  - ETH (Ethereum)
  - SOL (Solana)
  - BNB (Binance Coin)
  - ADA (Cardano)
  - XRP (Ripple)
  - DOGE (Dogecoin)
  - DOT (Polkadot)
  - MATIC (Polygon)
  - AVAX (Avalanche)
- Настройки сохраняются автоматически
- Минимум один тикер должен быть выбран

### 4. Кнопки управления
- **QUIT APP** - выход из приложения (красная кнопка)
- **ABOUT** - информация о приложении (синяя кнопка)
  - Версия приложения
  - Информация об авторе
  - Email контакта

## Технические детали

### Обновление данных
- Цены обновляются каждые 30 секунд
- Используется публичный API Binance (без аутентификации)
- Автоматическое обновление при изменении настроек

### Хранение данных
- Настройки сохраняются в UserDefaults
- Выбранные тикеры сохраняются автоматически
- При следующем запуске настройки восстанавливаются

### Интерфейс
- SwiftUI для всех окон
- WebKit для виджета Binance
- AppKit для меню бара
- Темная тема по умолчанию

## Использование

1. **Запуск приложения**
   - Приложение появляется в меню баре
   - Начинается автоматическое обновление цен

2. **Просмотр виджета**
   - Левая кнопка мыши на цены → открыть виджет Binance
   - Левая кнопка мыши еще раз → закрыть виджет

3. **Настройки**
   - Правая кнопка мыши на цены → открыть настройки
   - Выберите тикеры для отображения
   - Изменения применяются сразу

4. **Выход**
   - Правая кнопка мыши → Настройки → QUIT APP
   - Или через меню приложения

## Требования

- macOS 12.0 или выше
- Интернет-соединение для получения данных с Binance API
- Разрешение на доступ в интернет (автоматически запрашивается)
