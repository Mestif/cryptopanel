# CryptoPanel

Приложение для отслеживания криптовалют с использованием Binance API.

## Автор
**Mestif** (Mestif@gmail.com)

## Описание
Приложение получает данные о криптовалютах напрямую с биржи Binance и отображает их в удобном интерфейсе в виде меню бар приложения.

## Особенности
- Получение данных напрямую с Binance API
- Отображение топ-50 криптовалют по объему торгов
- Показ цены, изменения за 24 часа и объема торгов
- Автоматическое обновление данных
- Минималистичный интерфейс в меню бар

## Требования
- macOS 12.0 или выше
- Xcode 14.0 или выше
- Swift 5.9 или выше

## Установка и сборка

1. Откройте проект в Xcode:
   ```bash
   open CryptoPanelApp.xcodeproj
   ```

2. Выберите схему сборки "CryptoPanelApp"

3. Нажмите ⌘R для запуска или ⌘B для сборки

## Использование

После запуска приложение появится в меню бар (в правом верхнем углу экрана). 
Кликните на иконку биткоина, чтобы открыть панель с данными о криптовалютах.

## API

Приложение использует публичный API Binance:
- Endpoint: `https://api.binance.com/api/v3/ticker/24hr`
- Документация: https://binance-docs.github.io/apidocs/spot/en/#24hr-ticker-price-change-statistics
