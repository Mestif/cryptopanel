# Обновление версии 1.4.0

## Изменения

### 1. Поддержка длинных тикеров
- ✅ Убрано ограничение на длину тикера (было максимум 10 символов)
- ✅ Теперь можно добавлять тикеры любой длины (например: TRADOOR)
- ✅ Валидация только на буквы и цифры (без ограничения длины)

### 2. Ограничение на максимум 5 тикеров в панели
- ✅ Добавлена проверка: максимум 5 тикеров можно добавить в панель
- ✅ При попытке добавить 6-й тикер показывается предупреждение
- ✅ Можно снять галочку с одного тикера, чтобы добавить другой
- ✅ Ограничение применяется при добавлении и включении тикеров

### 3. Бегущая строка вместо таблицы
- ✅ Виджет Binance изменен с `layout="table"` на `layout="banner"`
- ✅ По левой кнопке мыши открывается бегущая строка с ценами
- ✅ Размер окна изменен с 800x400 на 800x100 пикселей
- ✅ Убраны лишние атрибуты виджета (coin-header, price-header и т.д.)

## Технические детали

- Убрана проверка `ticker.count <= 10`

- Добавлена проверка при включении тикера

### SettingsManager
- Функция `toggleTicker()` проверяет лимит 5 тикеров
- Функция `addTicker()` проверяет лимит перед автоматическим включением

### BinanceWidgetView
- Изменен `data-layout` с `"table"` на `"banner"`
- Убраны атрибуты: `data-coin-header`, `data-price-header`, `data-change-header`, `data-chart-header`
- Размер окна: 800x100 (было 800x400)

### AppDelegate
- Размер popover изменен на 800x100

## Использование

### Добавление длинных тикеров
1. Откройте настройки (правая кнопка мыши)
2. Введите любой тикер (например: TRADOOR, LINK, UNISWAP)
3. Нажмите "+" или Enter
4. Тикер будет добавлен (если не превышен лимит 5 тикеров)

### Ограничение на 5 тикеров
- В панели может быть максимум 5 тикеров одновременно
- Если уже 5 тикеров выбрано, нужно снять галочку с одного, чтобы добавить другой
- При попытке добавить 6-й тикер появится предупреждение

### Бегущая строка
- Левая кнопка мыши на цены → открывается бегущая строка
- Показывает цены BTC, ETH, SOL в виде бегущей строки
- Размер окна: 800x100 пикселей
- Компактное отображение

## Примеры

### Добавление длинного тикера
1. Введите "TRADOOR" в поле ввода
2. Нажмите "+"
3. TRADOOR будет добавлен в список пользовательских тикеров
4. Автоматически включится (если меньше 5 тикеров)

### Управление лимитом тикеров
- Если выбрано 5 тикеров: BTC, ETH, SOL, LINK, UNI
- Чтобы добавить TRADOOR, нужно снять галочку с одного из тикеров
- Затем можно добавить TRADOOR и включить его
