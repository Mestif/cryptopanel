# Инструкция по добавлению иконки

## Расположение иконки

Иконка должна называться `Icon.png` и находиться в одной из следующих локаций:

1. **В корне проекта:** `CryptoPanelApp/Icon.png`
2. **В папке с исходным кодом:** `CryptoPanelApp/CryptoPanelApp/Icon.png`

## Требования к иконке

- **Формат:** PNG
- **Имя файла:** `Icon.png` (с заглавной I)
- **Рекомендуемый размер:** 512x512 пикселей или больше
- **Формат для macOS:** PNG с альфа-каналом

## Автоматическая обработка

Скрипт сборки (`build.sh`) автоматически найдет и скопирует иконку в правильное место:
- Иконка будет скопирована в `build/CryptoPanelApp.app/Contents/Resources/Icon.png`
- Info.plist уже настроен для использования иконки

## Если иконка не найдена

Если иконка не найдена при сборке, приложение будет использовать стандартную иконку macOS. Это не критично, но рекомендуется добавить свою иконку.

## Создание иконки

Вы можете создать иконку используя:
- Xcode (File → New → File → Asset Catalog → App Icon)
- Онлайн инструменты для создания иконок
- Графические редакторы (Photoshop, GIMP, etc.)

## Проверка

После сборки проверьте наличие иконки:
```bash
ls -lh build/CryptoPanelApp.app/Contents/Resources/Icon.png
```

Если файл существует, иконка будет отображаться в Finder и Dock.
