# Обновление версии 1.2.0

## Новые возможности

### 1. Поддержка иконки приложения
- Добавлена поддержка иконки `Icon.png`
- Иконка автоматически копируется при сборке
- Info.plist настроен для использования иконки
- Подробные инструкции в `ICON_INSTRUCTIONS.md`

### 2. Ручной ввод тикеров
- Добавлено поле для ввода произвольных тикеров
- Можно ввести любой тикер вручную (например: LINK, UNI, AAVE)
- Валидация ввода (только буквы и цифры, максимум 10 символов)
- Пользовательские тикеры отображаются отдельно от предустановленных
- Можно удалить пользовательские тикеры кнопкой с крестиком

### 3. Улучшенный интерфейс настроек
- Разделение на "Предустановленные" и "Пользовательские" тикеры
- Увеличен размер окна настроек (550px высота)
- Улучшена визуальная структура

## Технические изменения

### StatusBarManager
- Добавлена функция `getBinanceSymbol()` для автоматического формирования пары Binance
- Поддержка произвольных тикеров (автоматически добавляется USDT)
- Если тикер есть в маппинге, используется он, иначе формируется как `{TICKER}USDT`

### SettingsManager
- Добавлена функция `addTicker()` для добавления тикеров
- Добавлена функция `removeTicker()` для удаления тикеров
- Валидация при добавлении тикеров

### SettingsView
- Добавлено текстовое поле для ввода тикера
- Кнопка добавления тикера
- Отображение пользовательских тикеров с возможностью удаления
- Обработка ошибок при вводе

### build.sh
- Автоматический поиск и копирование иконки
- Поддержка иконки в корне проекта или в папке CryptoPanelApp

## Использование

### Добавление иконки
1. Поместите файл `Icon.png` в папку `CryptoPanelApp/`
2. Пересоберите приложение: `./build.sh`
3. Иконка будет автоматически включена в приложение

### Добавление произвольного тикера
1. Откройте настройки (правая кнопка мыши на цены)
2. Введите тикер в поле "Добавить тикер вручную"
3. Нажмите кнопку "+" или Enter
4. Тикер появится в списке "Пользовательские"
5. Цена будет автоматически обновляться

### Удаление пользовательского тикера
1. Откройте настройки
2. Найдите тикер в разделе "Пользовательские"
3. Нажмите на красный крестик рядом с тикером
4. Тикер будет удален (если это не последний тикер)

## Примеры использования

### Добавление популярных тикеров
- LINK (Chainlink)
- UNI (Uniswap)
- AAVE (Aave)
- MKR (Maker)
- COMP (Compound)
- SNX (Synthetix)

Все эти тикеры будут автоматически преобразованы в пары Binance (например, LINK → LINKUSDT).

## Важные замечания

1. **Валидация тикеров:** Тикер должен содержать только буквы и цифры, максимум 10 символов
2. **Минимум один тикер:** Нельзя удалить последний тикер
3. **Формат Binance:** Все тикеры автоматически преобразуются в формат `{TICKER}USDT`
4. **Иконка опциональна:** Приложение будет работать и без иконки, но рекомендуется добавить свою
