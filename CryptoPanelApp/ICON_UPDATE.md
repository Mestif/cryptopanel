# Обновление иконки приложения

## Иконка применена

### Текущая иконка
- **Файл:** `ICON.png` (563 KB, 541x617 пикселей)
- **Формат:** PNG с прозрачностью (RGBA)
- **Описание:** Иконка с текстом "CRYPTO PRICES by Mestif", графиком роста и символами BTC и $

### Где отображается иконка

1. **Finder**
   - Иконка приложения в списке файлов
   - Автоматически применяется из Resources/Icon.png

2. **Dock**
   - Иконка в Dock при запуске приложения
   - Используется системная иконка приложения

3. **Меню приложений**
   - Иконка в списке запущенных приложений
   - Используется системная иконка

4. **Окно About**
   - Иконка отображается в диалоговом окне About
   - Загружается из Resources/Icon.png
   - Масштабируется до 128x128 для отображения

### Как обновить иконку

1. Замените файл `ICON.png` в корне проекта `CryptoPanelApp/`
2. Пересоберите приложение: `./build.sh`
3. Иконка автоматически скопируется в `build/CryptoPanelApp.app/Contents/Resources/Icon.png`

### Требования к иконке

- **Формат:** PNG
- **Размер:** Рекомендуется 512x512 или больше (квадратная)
- **Имя файла:** `ICON.png` или `Icon.png`
- **Расположение:** В корне проекта `CryptoPanelApp/`

### Проверка

После сборки проверьте:
```bash
ls -lh build/CryptoPanelApp.app/Contents/Resources/Icon.png
```

Файл должен существовать и иметь размер около 563 KB (или размер вашей иконки).
