# Распространение приложения CryptoPanel

## DMG файл создан

### Файл для распространения
- **Имя файла:** `CryptoPanel_v1.0.0.dmg`
- **Размер:** ~1.4 MB (сжатый)
- **Формат:** DMG (Disk Image) для macOS

### Содержимое DMG
- `CryptoPanelApp.app` - приложение
- `Applications` - символическая ссылка для перетаскивания приложения

### Как использовать DMG

1. **Откройте DMG файл:**
   - Дважды кликните на `CryptoPanel_v1.0.0.dmg`
   - DMG откроется в Finder

2. **Установка приложения:**
   - Перетащите `CryptoPanelApp.app` в папку `Applications`
   - Или скопируйте приложение в любую папку

3. **Запуск:**
   - Найдите `CryptoPanelApp` в Applications
   - Запустите приложение
   - При первом запуске macOS может запросить разрешение

### Распространение

DMG файл готов к распространению:
- Можно загрузить на сайт
- Можно отправить по email
- Можно разместить в облачном хранилище
- Можно подписать для Gatekeeper (опционально)

### Безопасность

При первом запуске macOS может показать предупреждение:
- Это нормально для неподписанных приложений
- Пользователь может разрешить запуск в настройках безопасности
- Для устранения предупреждения нужно подписать приложение сертификатом Apple Developer

### Создание DMG

Для создания нового DMG файла:
```bash
cd CryptoPanelApp
./create_dmg.sh
```
