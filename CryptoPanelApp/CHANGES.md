# Изменения в версии 1.1.0

## Новые возможности

### 1. Отображение цен в меню баре
- Вместо иконки теперь отображаются цены BTC, ETH и SOL
- Формат: `BTC: $XX,XXX | ETH: $X,XXX | SOL: $XXX`
- Автоматическое обновление каждые 30 секунд

### 2. Виджет Binance (левая кнопка мыши)
- По левому клику на цены открывается виджет Binance
- Отображает таблицу с BTC, ETH, SOL
- Показывает цену, изменение за 24ч и график
- Размер: 800x600 пикселей

### 3. Меню настроек (правая кнопка мыши)
- По правому клику открывается меню настроек
- Выбор тикеров для отображения в меню баре
- Доступные тикеры: BTC, ETH, SOL, BNB, ADA, XRP, DOGE, DOT, MATIC, AVAX
- Настройки сохраняются автоматически

### 4. Кнопки управления
- **QUIT APP** - выход из приложения
- **ABOUT** - информация о приложении

## Технические изменения

- Добавлен `StatusBarManager` для управления ценами в меню баре
- Добавлен `BinanceWidgetView` с WebView для виджета Binance
- Добавлен `SettingsView` для настроек приложения
- Обновлен `AppDelegate` для обработки левого и правого кликов мыши

## Использование

1. **Левая кнопка мыши** на цены → открыть виджет Binance
2. **Правая кнопка мыши** на цены → открыть настройки
3. В настройках можно выбрать какие тикеры показывать
4. QUIT APP для выхода, ABOUT для информации

## Файлы

- `StatusBarManager.swift` - управление ценами в меню баре
- `BinanceWidgetView.swift` - виджет Binance
- `SettingsView.swift` - настройки приложения
- `AppDelegate.swift` - обновлен для новых функций
