# Обновление версии 1.8.0

## Изменения

### 1. Скрыты надписи "Powered by binance" и "Disclaimer" из виджета
- ✅ Добавлены CSS стили для скрытия элементов с текстом "Powered by" и "Disclaimer"
- ✅ Добавлен JavaScript для динамического скрытия этих элементов
- ✅ Элементы скрываются через:
  - `display: none`
  - `visibility: hidden`
  - `opacity: 0`
  - `height: 0` и `width: 0`
- ✅ Периодическое обновление каждые 500ms для надежного скрытия
- ✅ Виджет теперь показывает только бегущую строку с ценами

### 2. Применена новая иконка приложения
- ✅ Иконка из `ICON.png` (541x617, PNG) скопирована в Resources
- ✅ Иконка содержит логотипы BTC, ETH, SOL
- ✅ Автоматически применяется при сборке
- ✅ Отображается в Finder, Dock и меню приложений

## Технические детали

### BinanceWidgetView
- Добавлены CSS селекторы для скрытия:
  - Элементы с классами содержащими "powered" или "disclaimer"
  - Элементы с ID содержащими "powered" или "disclaimer"
  - Ссылки на Binance или disclaimer
  - Элементы с текстом "Powered by" или "Disclaimer"
- Добавлена JavaScript функция `hideBranding()`:
  - Поиск всех элементов с текстом "Powered by" или "Disclaimer"
  - Скрытие через множественные CSS свойства
  - Обработка ссылок на Binance
  - Периодическое обновление каждые 500ms

### build.sh
- Поддержка `ICON.png` (с заглавными буквами)
- Иконка копируется как `Icon.png` в Resources
- Приоритет поиска: ICON.png → Icon.png

## Использование

### Виджет
- Виджет теперь показывает только бегущую строку с ценами
- Надписи "Powered by binance" и "Disclaimer" полностью скрыты
- Более чистый и минималистичный интерфейс

### Иконка
- Новая иконка с логотипами BTC, ETH, SOL
- Размер: 541x617 пикселей
- Формат: PNG с прозрачностью
- Автоматически применяется при сборке

## Файлы

- `BinanceWidgetView.swift` - CSS и JavaScript для скрытия надписей
- `build.sh` - копирование ICON.png
- `ICON.png` - новая иконка приложения (541x617)

## Примечания

Если надписи все еще видны:
- Виджет Binance может загружаться асинхронно
- JavaScript применяется с задержкой 1 секунда после загрузки
- Обновление происходит каждые 500ms для надежного скрытия
- Если проблема сохраняется, возможно виджет использует iframe - в этом случае нужно дополнительное решение
