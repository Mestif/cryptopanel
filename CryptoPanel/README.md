# CryptoPanel

Приложение для отслеживания криптовалют с использованием Binance API.

## Автор
Mestif (Mestif@gmail.com)

## Описание
Приложение получает данные о криптовалютах напрямую с биржи Binance и отображает их в удобном интерфейсе.

## Требования
- macOS 12.0 или выше
- Xcode 14.0 или выше

## Установка
1. Откройте проект в Xcode
2. Выберите схему сборки
3. Нажмите Run
